# WearSource

## Purpose

Tracks how a wear event was created.

---

## Values

### DASHBOARD

Marked as worn from Dashboard.

### CALENDAR

Created from Calendar flow.

### OUTFIT_PAGE

Created from Outfit page.

### MANUAL

Created manually by user.
