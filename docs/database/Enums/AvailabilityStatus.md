# AvailabilityStatus

## Purpose

Indicates whether a wardrobe item can currently be worn.

---

## Values

### AVAILABLE

Item is ready to wear.

### LAUNDRY

Item is currently being washed.

### UNAVAILABLE

Item cannot be worn.

---

## Used By

- [[WardrobeItem]]

---

## Dashboard Usage

- Needs Attention
- Outfit Validation
- Readiness Checks
