## Outfit Planning

As a user,
I want to know what to wear tomorrow,
so that I don't waste time in the morning.

## Wardrobe Management

As a user,
I want to track the availability of my clothes,
so that I don't plan outfits using unavailable items.

## Clothing Analytics

As a user,
I want to know which clothes are used most often,
so that I can optimize my wardrobe.

## Conscious Shopping

As a user,
I want to understand whether a new purchase adds value,
so that I avoid unnecessary spending.
